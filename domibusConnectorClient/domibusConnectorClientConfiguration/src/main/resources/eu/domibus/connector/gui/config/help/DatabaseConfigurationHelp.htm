Database Configuration Help

 

Database Dialect

The dialect value depends on the database system that is in use.

Here some examples:

 -Oracle: org.hibernate.dialect.Oracle10gDialect

 -MySQL : org.hibernate.dialect.MySQLDialect

 

JDBC DriverClass Name

The driverClassName value depends on the database system and the JDBC driver that is in use.

Here some examples:

 -Oracle: oracle.jdbc.driver.OracleDriver

 -MySQL : com.mysql.jdbc.Driver

 

Database Connection URL

The JDBC URL to your database.

Example: jdbc:oracle:thin:@servername:port:instancename

 

Schema Username

The username to your database schema given by your database administrator.

 

Schema Password

The password to your database schema given by your database administrator.

 

 

ConnectionPool acquireIncrement

Defines how many connections should be acquired when pool needs to be extended.

 

ConnectionPool minPoolSize

Defines the minimum connections the connection pool must have.

 

ConnectionPool maxPoolSize

Defines the maximum connections the connection pool may have.

 

ConnectionPool maxIdleTime

Defines after what time period (in seconds) a connection expires when not used actively.
